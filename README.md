# afi-config

**Configuration hub for AFI Protocol**

This repository contains network configurations, persona/agent registries, validator/mentor registries, and .afi-codex schemas.

## Contents

- **schemas/** - JSON schemas for blueprints, pipelines, and manifests
- **templates/** - Configuration templates and construct patterns
- **cli_utils/** - CLI utilities for config validation

## Purpose

afi-config is the single source of truth for all AFI configuration:
- Network configurations
- Persona and agent registries
- Validator and mentor registries
- Pipeline and DAG schemas
- Codex schemas
